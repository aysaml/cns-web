System Description 系统描述

辽宁石油化工大学校园导航系统（LNPU-CNS），旨在为学生提供一套方便完善的校园导航、建筑物介绍、周边美食推荐、周边文化景点游览服务。通过技术手段，可以自动生成特定地点的二维码，用户通过扫描二维码，可以展示该地点的名称，位置，详细描述和导游信息。同时，用户可以通过访问系统网站，使用地点接入、热点推荐、导游报名、公告板、留言等功能。通过本系统，可以极大的方便、丰富师生的校园生活，同时对于来访人员更好的介绍本校的历史文化底蕴和学风建设成果等。
